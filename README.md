- [ ]  (10 puntos) Por no incluir ni un solo archivo de html en su proyecto
- [x]  (5 puntos) Por implementar React
- [x]  (15 puntos) Por implementar ESLint con las reglas:

  'airbnb-base',
  'airbnb/rules/react',
